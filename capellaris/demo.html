Chat com Agentes

Esta demonstração simula uma interação com diferentes agentes especializados através de uma interface de chat. Você pode ver como os agentes respondem a diferentes tipos de solicitações em suas áreas de especialidade.

Simulação de Chat com Agente de Marketing

Usuário:

@agente_marketing analise os resultados da nossa campanha de email marketing do último trimestre e sugira melhorias para aumentar a taxa de conversão.

Agente de Marketing:

Vou analisar os resultados da campanha de email marketing do último trimestre e sugerir melhorias para aumentar a taxa de conversão. Para isso, precisarei acessar os dados de desempenho da campanha. Você confirma que posso prosseguir com esta análise?

Usuário:

Aprovado.

Agente de Marketing:

Aqui está a análise dos resultados da campanha de email marketing do último trimestre:

Métricas Principais:

  • Taxa de abertura: 24.3% (↓ 2.1% em relação ao trimestre anterior)
  • Taxa de clique: 3.7% (↓ 0.5% em relação ao trimestre anterior)
  • Taxa de conversão: 1.2% (↓ 0.3% em relação ao trimestre anterior)
  • Taxa de descadastramento: 0.8% (↑ 0.2% em relação ao trimestre anterior)

Análise:

Observamos uma tendência de queda em todos os indicadores principais de desempenho. A análise dos dados sugere que os principais fatores contribuintes são:

  • Assuntos de email menos eficazes (taxa de abertura em declínio)
  • Conteúdo menos envolvente (taxa de clique em declínio)
  • Páginas de destino não otimizadas (taxa de conversão em declínio)
  • Frequência excessiva de emails (aumento na taxa de descadastramento)

Recomendações para melhorar a taxa de conversão:

  1. Melhorar a segmentação: Refinar os segmentos de público com base no comportamento de compra e engajamento anterior para enviar conteúdo mais relevante.
  2. Otimizar assuntos de email: Realizar testes A/B com diferentes formatos de assunto para identificar o que gera maiores taxas de abertura.
  3. Redesenhar CTAs: Tornar os botões de call-to-action mais visíveis e convincentes, com mensagens claras e urgentes.
  4. Melhorar páginas de destino: Otimizar as páginas de destino para que sejam mais alinhadas com a promessa do email e facilitar o processo de conversão.
  5. Ajustar frequência: Reduzir a frequência geral de emails e implementar uma estratégia de envio baseada nas preferências individuais dos usuários.

Gostaria que eu elaborasse alguma dessas recomendações em mais detalhes ou que preparasse um plano de ação específico para implementá-las?

Criação de um Agente Personalizado

from gerenciador_agentes_ia.core import AgentManager
from gerenciador_agentes_ia.agents import Agent

# Inicializar o gerenciador de agentes
agent_manager = AgentManager()

# Criar um agente personalizado para análise de dados
data_agent = Agent(
    id="agent_data_analysis",
    name="Agente de Análise de Dados",
    description="Especialista em análise e visualização de dados",
    specialty="data_analysis",
    base_prompt="""
    Você é um especialista em análise de dados com ampla experiência em estatística,
    visualização de dados e interpretação de resultados. Seu objetivo é ajudar a extrair
    insights valiosos de conjuntos de dados e apresentá-los de forma clara e acionável.
    
    Ao analisar dados, você deve:
    1. Identificar tendências, padrões e anomalias
    2. Aplicar métodos estatísticos apropriados
    3. Criar visualizações informativas
    4. Fornecer interpretações claras dos resultados
    5. Sugerir ações baseadas nos insights
    
    Suas respostas devem ser estruturadas, precisas e focadas em insights acionáveis.
    """,
    tools=[
        "data_import",
        "statistical_analysis",
        "data_visualization",
        "report_generation"
    ],
    config={
        "model": "gpt-4",
        "temperature": 0.2,
        "max_tokens": 2000,
        "supported_formats": ["csv", "excel", "json", "sql"]
    }
)

# Registrar o agente no sistema
agent_manager.register_agent(data_agent)

# Verificar se o agente foi registrado corretamente
registered_agent = agent_manager.get_agent("agent_data_analysis")
print(f"Agente registrado: {registered_agent.name}")

Integração com API Externa

from gerenciador_agentes_ia.core import ToolRegistry
from gerenciador_agentes_ia.tools import Tool
import requests

# Criar uma ferramenta para integração com API de clima
class WeatherTool(Tool):
    def __init__(self, api_key):
        super().__init__(
            name="weather_tool",
            description="Obtém informações de clima para uma localização específica",
            parameters={
                "location": {
                    "type": "string",
                    "description": "Nome da cidade ou coordenadas (latitude,longitude)"
                },
                "units": {
                    "type": "string",
                    "description": "Unidades de medida (metric, imperial)",
                    "default": "metric"
                }
            }
        )
        self.api_key = api_key
        self.base_url = "https://api.openweathermap.org/data/2.5/weather"
    
    def execute(self, parameters):
        """
        Executa a ferramenta com os parâmetros fornecidos.
        
        Args:
            parameters (dict): Parâmetros para a execução
            
        Returns:
            dict: Resultado da execução
        """
        try:
            # Preparar parâmetros da requisição
            params = {
                "q": parameters.get("location"),
                "units": parameters.get("units", "metric"),
                "appid": self.api_key
            }
            
            # Fazer requisição à API
            response = requests.get(self.base_url, params=params)
            response.raise_for_status()
            
            # Processar resposta
            data = response.json()
            
            # Formatar resultado
            result = {
                "location": f"{data['name']}, {data['sys']['country']}",
                "temperature": {
                    "current": data["main"]["temp"],
                    "feels_like": data["main"]["feels_like"],
                    "min": data["main"]["temp_min"],
                    "max": data["main"]["temp_max"]
                },
                "humidity": data["main"]["humidity"],
                "pressure": data["main"]["pressure"],
                "wind": {
                    "speed": data["wind"]["speed"],
                    "direction": data["wind"]["deg"]
                },
                "weather": {
                    "main": data["weather"][0]["main"],
                    "description": data["weather"][0]["description"]
                }
            }
            
            return {
                "status": "success",
                "data": result
            }
            
        except Exception as e:
            return {
                "status": "error",
                "message": str(e)
            }

# Registrar a ferramenta no sistema
tool_registry = ToolRegistry()
weather_tool = WeatherTool(api_key="your_api_key_here")
tool_registry.register_tool(weather_tool)

# Atribuir a ferramenta a um agente
agent_manager = AgentManager()
agent = agent_manager.get_agent("agent_operations_1")
agent.add_tool("weather_tool")

Criação de um Squad Personalizado

from gerenciador_agentes_ia.core import SquadManager, AgentManager
from gerenciador_agentes_ia.squads import Squad, WorkflowStep, WorkflowType

# Inicializar os gerenciadores
agent_manager = AgentManager()
squad_manager = SquadManager(agent_manager)

# Definir os passos do fluxo de trabalho para análise de mercado
workflow_steps = [
    WorkflowStep(
        agent_id="agent_data_analysis",
        task="data_collection",
        description="Coletar e preparar dados de mercado"
    ),
    WorkflowStep(
        agent_id="agent_marketing_1",
        task="trend_analysis",
        description="Analisar tendências e comportamento do consumidor"
    ),
    WorkflowStep(
        agent_id="agent_financial_1",
        task="financial_analysis",
        description="Realizar análise financeira do mercado"
    ),
    WorkflowStep(
        agent_id="agent_sales_1",
        task="competitor_analysis",
        description="Analisar concorrentes e oportunidades"
    ),
    WorkflowStep(
        agent_id="agent_marketing_1",
        task="consolidation",
        description="Consolidar os resultados e preparar relatório final"
    )
]

# Criar um squad personalizado para análise de mercado
market_analysis_squad = Squad(
    id="squad_market_analysis_custom",
    name="Squad de Análise de Mercado Personalizado",
    description="Equipe especializada em análise completa de mercados e segmentos",
    agents=[
        {"id": "agent_data_analysis", "role": "data_analyst"},
        {"id": "agent_marketing_1", "role": "marketing_specialist"},
        {"id": "agent_financial_1", "role": "financial_analyst"},
        {"id": "agent_sales_1", "role": "sales_strategist"}
    ],
    leader_id="agent_marketing_1",
    workflow={
        "type": WorkflowType.SEQUENTIAL,
        "steps": workflow_steps
    },
    config={
        "max_execution_time": 3600,
        "intermediate_results": True,
        "approval_required": True,
        "output_format": "report"
    }
)

# Registrar o squad no sistema
squad_manager.register_squad(market_analysis_squad)

# Executar uma tarefa com o squad
task_result = squad_manager.execute_task(
    squad_id="squad_market_analysis_custom",
    task="Realize uma análise completa do mercado de software de gestão empresarial para pequenas e médias empresas no Brasil, incluindo análise de concorrentes, tendências e oportunidades.",
    context={
        "target_market": "Pequenas e médias empresas",
        "geographic_focus": "Brasil",
        "industry_sectors": ["Varejo", "Serviços", "Manufatura"],
        "time_horizon": "12 meses"
    },
    requester="user@company.com"
